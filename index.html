CoffeeFx

CoffeeFx is a small JavaScript library making CSS3 backed animation extremely simple and elegant.

Examples

Watch our clock!!! clock

CoffeeFx#set(prop, val)

Set prop to val.

Play 
          
coffeefx('#example-1 .box')
  .set('margin-left', 200)
  .end();

CoffeeFx#add(prop, val)

Increment prop by val, where val is an Number (click several times).

Play 
          
coffeefx('#example-2 .box')
  .add('margin-left', 200)
  .end();

CoffeeFx#sub(prop, val)

Decrement prop by val, where val is an Number (click several times).

Play 
          
coffeefx('#example-3 .box')
  .sub('margin-left', 100)
  .end();

CoffeeFx#rotate(deg)

Rotate by deg.

Play 
          
coffeefx('#example-4 .box')
  .rotate(140)
  .end();

CoffeeFx#duration(n)

Set animation duration to n which is a Number or a string such as "4s".

Play 
          
coffeefx('#example-5 .box')
  .set('background-color', 'blue')
  .transitionDuration('2s')
  .end();

CoffeeFx#translate(x[, y])

Translate x and optionally y axis. Aliased as CoffeeFx#to(x[, y]).

Play 
          
coffeefx('#example-6 .box')
  .translate(300, 80)
  .end();

CoffeeFx#x(n) / CoffeeFx#y(n)

Translate x or y axis. Aliased by CoffeeFx#translateX(n) and CoffeeFx#translateY(n).

Play 
          
coffeefx('#example-7 .box')
  .x(300)
  .y(20)
  .end();

CoffeeFx#skew(x[, y])

Skew x, and optionally y. CoffeeFx#skewX(n) and CoffeeFx#skewY(n) are also available.

Play 
          
coffeefx('#example-8 .box')
  .x(300)
  .skew(50)
  .set('height', 20)
  .end();

CoffeeFx#scale(x[, y])

Scale the x, and optionally y axis. CoffeeFx#scaleX(n) and CoffeeFx#scaleY(n) are also available.

Play 
          
coffeefx('#example-9 .box')
  .scale(3)
  .end();

CoffeeFx#ease(fn)

Use the given easing fn.

Play
default
in
out
in-out
snap
(0,1,1,0)
          
coffeefx('#example-10 .box1').x(400).end();
coffeefx('#example-10 .box2').ease('in').x(400).end();
coffeefx('#example-10 .box3').ease('out').x(400).end();
coffeefx('#example-10 .box4').ease('in-out').x(400).end();
coffeefx('#example-10 .box5').ease('snap').x(400).end();
coffeefx('#example-10 .box6').ease('cubic-bezier(0,1,1,0)').x(400).end();

setTimeout(function(){
  coffeefx('#example-10 .box1').x(0).end();
  coffeefx('#example-10 .box2').x(0).end();
  coffeefx('#example-10 .box3').x(0).end();
  coffeefx('#example-10 .box4').x(0).end();
  coffeefx('#example-10 .box5').x(0).end();
  coffeefx('#example-10 .box6').x(0).end();
}, 1200);

CoffeeFx#end([fn])

The end() method triggers the animation to play, optionally invoking the callback fn when complete.

Play 
          
coffeefx('#example-11 .box')
  .set('background-color', 'red')
  .transitionDuration(1000)
  .end(function(){
    coffeefx('#example-11 .box')
      .set('background-color', 'white')
      .end();
  });

CoffeeFx#delay(n)

Set animation delay to n which is a Number or a string such as "4s".

Play 
          
coffeefx('#example-12 .box')
  .set('background-color', 'blue')
  .delay('2s')
  .end();

CoffeeFx#then([fn])

Defer an action such as invoking a fn, end()ing the given CoffeeFx instance, or returning a clone for chaining. The CoffeeFx#pop() method is used to return the current CoffeeFx instance's parent in the chain.

Play 
          
var coffeefxBack = coffeefx('#example-13 .box')
  .set('background-color', 'white')
  .x(0);

coffeefx('#example-13 .box')
  .set('background-color', 'red')
  .x(500)
  .then(coffeefxBack)
  .end();

coffeefx('#example-13 .box2')
  .set('background-color', 'red')
  .x(500)
  .scale(.5)
  .rotate(60)
    .then()
      .rotate(30)
      .scale(1.5)
      .set('border-radius', 5)
      .set('background-color', 'white')
      .then()
        .set('opacity', 0)
        .pop()
      .pop()
  .end();

coffeefx.select(selector)

This function is used throughout coffeefx to select elements. For example if we wanted to utilize jQuery, we could re-define this function as shown below.

          
coffeefx.select = function(selector){
  return $(selector).get(0);
};

coffeefx.defaults

Defaults used throughout CoffeeFx, simply re-define to apply a new default.

          
coffeefx.defaults = {
  duration: 500
};

coffeefx.ease

Easing function map, used by CoffeeFx#ease() to allow for coffeefx('foo').ease('in') etc.

          
coffeefx.ease = {
    'in': 'ease-in'
  , 'out': 'ease-out'
  , 'in-out': 'ease-in-out'
  , 'snap': 'cubic-bezier(0,1,.5,1)'
};

coffeefx.version

The library version in the form "MAJOR.MINOR.PATCH".

          
coffeefx.version = "n.n.n";
